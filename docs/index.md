# punkst

punkst provides a set of tools for analyzing high resolution spatial transcriptomics data.

Check the [Installation](install.md) page to install punkst.

Check the [Input](input/index.md) page for help on preparing your data from vairous platforms.

Check [Quickstart](workflows/index.md) to run a basic [pixel level factor analysis](https://doi.org/10.1038/s41592-024-02415-2) pipeline on the example data, and [Basics](workflows/basics.md) for details on the workflow.

See [Modules](modules/index.md) for available operations and downstream analyses.

See [Deployment](workflows/deploy_carto.md) for packaging your data and results for visualization on [CartoScope](https://main.cartoscope.app/summary/).
